# Side navigation

*This pattern page is coming soon. It will document side navigation best practices and provide ready-to-copy examples.*

## When to use

- **Primary navigation** for applications
- **Hierarchical content** organization
- **Persistent navigation** across pages

## When not to use

- **Secondary navigation** (use tabs or breadcrumbs)
- **Mobile navigation** (use responsive patterns)

## Composition

Built from atoms:
- `EuiSideNav` (container)
- `EuiSideNavItem` (navigation items)
- `EuiIcon` (navigation icons)
- `EuiBadge` (item indicators)

## Variants

- **Basic side nav** - Simple navigation structure
- **Nested navigation** - Hierarchical organization
- **Collapsible sections** - Expandable navigation
- **Contextual nav** - Dynamic navigation items
