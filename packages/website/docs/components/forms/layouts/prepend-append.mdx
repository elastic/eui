# Prepends and appends

Most form controls accept a `prepend` and `append` prop that allows passing a string or single node. These are great for demarcating the input's metric like "px" or "ms". 

Use the dedicated **EuiFormPrepend** and **EuiFormAppend** components to pass the content and ensure expected styling.
Where needed, you can wrap those components with other container components like [**EuiPopover**](../../containers/popover.mdx) or [**EuiToolTip**](../../display/tooltip.mdx).
Simple string content will automatically be wrapped into **EuiFormPrepend** or **EuiFormAppend**.

You can render the `prepend` and `append` content as interactive button element by setting `element="button"` on **EuiFormPrepend** and **EuiFormAppend** and passing `onClick` as handler.

:::warning `prepend` and `append` only support single interactions.
:::


```tsx interactive
import React, { useState } from 'react';
import {
  EuiFlexGroup,
  EuiFormAppend,
  EuiFormPrepend,
  EuiButtonEmpty,
  EuiButtonIcon,
  EuiFieldText,
  EuiIcon,
  EuiIconTip,
  EuiPopover,
  EuiSpacer,
  EuiSwitch,
  EuiText,
  EuiToolTip,
  useEuiTheme,
} from '@elastic/eui';

export default () => {
  const [isCompressed, setCompressed] = useState(false);
  const [isDisabled, setDisabled] = useState(false);
  const [isReadOnly, setReadOnly] = useState(false);

  const { euiTheme } = useEuiTheme();

  const props = {
    compressed: isCompressed,
    disabled: isDisabled,
    readOnly: isReadOnly,
    placeholder: 'Text field'
  }

  return (
    <>
      <EuiFlexGroup responsive={false}>
        <EuiSwitch
          label="compressed"
          checked={isCompressed}
          onChange={(e) => setCompressed(e.target.checked)}
        />
        <EuiSwitch
          label="disabled"
          checked={isDisabled}
          onChange={(e) => setDisabled(e.target.checked)}
        />
        <EuiSwitch
          label="readOnly"
          checked={isReadOnly}
          onChange={(e) => setReadOnly(e.target.checked)}
        />
      </EuiFlexGroup>
      
      <EuiSpacer />

      <EuiFlexGroup direction="column" gutterSize="m">
        <EuiFieldText
          {...props}
          id="input-0"
          prepend="String"
          append={<EuiFormAppend label="String" iconLeft="gear" />}
          autoFocus
        />

        <EuiFieldText
          {...props}
          id="input-1"
          prepend={<EuiFormPrepend label="String" iconRight="gear" />}
          append={
            <EuiFormAppend
              element="button"
              label="String"
              iconLeft="gear"
              onClick={() => {}}
            />
          }
        />

        <EuiFieldText
          {...props}
          id="input-2"
          prepend={
            <EuiFormPrepend label="String" iconLeft="vector" iconRight="gear" />
          }
          append={
            <EuiFormAppend
              element="button"
              label="String"
              iconLeft="gear"
              iconRight="vector"
              onClick={() => {}}
            />
          }
        />

        <EuiFieldText
          {...props}
          id="input-3"
          prepend={
            <EuiFormPrepend label="String" iconLeft="vector" iconRight="gear">
              <EuiNotificationBadge color="subdued">1</EuiNotificationBadge>
            </EuiFormPrepend>
          }
          append={
            <EuiFormAppend
              element="button"
              label="String"
              iconLeft="gear"
              iconRight="vector"
              onClick={() => {}}
            >
              <EuiNotificationBadge>1</EuiNotificationBadge>
            </EuiFormAppend>
          }
        />

        <EuiFieldText
          {...props}
          id="input-4"
          prepend={
            <EuiToolTip content="content">
              <EuiFormPrepend label="Tooltip" />
            </EuiToolTip>
          }
          append={
            <EuiToolTip content="content">
              <EuiFormAppend
                element="button"
                label="Tooltip"
                onClick={() => {}}
              />
            </EuiToolTip>
          }
        />

        <EuiFieldText
          {...props}
          id="input-5"
          prepend={
            <EuiPopover
              button={
                 <EuiFormPrepend
                    element="button"
                    iconLeft="calendar"
                    iconRight="arrowDown"
                    onClick={() => {}}
                  />
              }
              closePopover={() => {}}
            />
          }
          append={
            <EuiPopover
              button={
                <EuiFormAppend
                  element="button"
                  iconRight="arrowDown"
                  label="Popover"
                  onClick={() => {}}
                />
              }
              closePopover={() => {}}
            />
          }
        />

        <EuiFieldText
          {...props}
          id="input-6"
          prepend={<EuiFormPrepend iconLeft="vector" />}
          append={
            <EuiFormAppend
              element="button"
              iconLeft="gear"
              onClick={() => {}}
            />
          }
        />
      </EuiFlexGroup>
    </>
  );
};
```

## Semantic label relations

:::accessibility HTML form elements (e.g. `<input>` or `<textarea>`) require a label for accessibility.
If you're wrapping your component in a **EuiFormRow** with `label` prop, the label is automatically linked to the wrapped form control. 
If your using a standalone form component (without wrapping** EuiFormRow)** you'll need to manually ensure the form control has an accessible label.
:::

To ensure the form control has an accessible label, you can pass `id` on **EuiFormControlLayout** (or a form control like **EuiFieldText** or **EuiFieldNumber** that renders a `EuiFormControlLayout` 
internally when using it with `controlOnly={false}`). This renders non-interactive `prepend` or `append` content automatically as semantically linked `<label>` element. You can also use `inputId` 
on **EuiFormPrepend** or **EuiFormAppend** to link the content as label to a specific form control. 

If you're using both `prepend` and `append` but don't want both automatically linked as label when passing `id`, use `inputId=""` on either **EuiFormPrepend** or **EuiFormAppend** to unset one of them.

import { Example } from '@site/src/components';
import { EuiFieldText, EuiFormAppend, EuiFormPrepend, EuiSpacer, EuiFormRow } from '@elastic/eui';

<Example>
  <Example.Preview>
    <EuiFieldText
      id="input-id"
      placeholder="Text field"
      prepend={<EuiFormPrepend label="Linked label" />}
      append={
        <EuiFormAppend
          label="Additional text"
          inputId=""
        />
      }
    />
  </Example.Preview>
  <Example.Snippet>
    ```ts
      <EuiFieldText
        id="input-id"
        placeholder="Text field"
        prepend={<EuiFormPrepend label="Linked label" />}
        append={
          <EuiFormAppend
            label="Additional text"
            inputId=""
          />
        }
      />
    ```
  </Example.Snippet>
</Example>


If you don't pass an `id`, the non-interactive `prepend` and `append` content will then be rendered as a simple, non-linked `<span>` element.
To ensure your form control has an accessible label anyway, pass an `aria-label` or `aria-labelledby` to the form control (e.g. **EuiFieldText** or **EuiFieldNumber**).

<Example>
 <Example.Preview>
    <EuiFieldText
      placeholder="Text field"
      prepend={<EuiFormPrepend label="Custom linked label" id="prepend-id" />}
      aria-labelledby="prepend-id"
    />

    <EuiSpacer size="m" />

    <EuiFieldText
      placeholder="Text field"
      prepend={<EuiFormPrepend label="Non-linked text" />}
      aria-label="Input label"
    />
  </Example.Preview>
  <Example.Snippet>
    ```ts
      <EuiFieldText
        placeholder="Text field"
        prepend={<EuiFormPrepend label="Custom linked label" id="prepend-id" />}
        aria-labelledby="prepend-id"
      />

      <EuiFieldText
        placeholder="Text field"
        prepend={<EuiFormPrepend label="Non-linked text" />}
        aria-label="Input label"
      />
    ```
  </Example.Snippet>
</Example>
