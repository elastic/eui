# Horizontal rule

**EuiHorizontalRule** is a styled `<hr>` element. It can be one of three provided sizes (lengths), by default it will be `full`.

```tsx interactive
import React from 'react';
import { EuiHorizontalRule, EuiCode } from '@elastic/eui';

export default () => (
  <div css={{ textAlign: "center" }}>
    <EuiCode>quarter</EuiCode>
    <EuiHorizontalRule size="quarter" />
    <EuiCode>half</EuiCode>
    <EuiHorizontalRule size="half" />
    <EuiCode>full (default)</EuiCode>
    <EuiHorizontalRule />
  </div>
);
```

## Margins

The spacing added before and after **EuiHorizontalRule** can be adjusted using the `margin` prop. Donâ€™t forget that margins will collapse against items that proceed and/or follow.

```tsx interactive
import React from 'react';
import { EuiHorizontalRule, EuiCode } from '@elastic/eui';

const Container = ({ children }) => (
  <div css={({ euiTheme }) => ({
    padding: 0.01,
    marginBottom: euiTheme.size.base,
    background: '#047471', /* TODO replace with euiTheme.colors.??? */
    '--ifm-hr-margin-vertical': 0, /* override docusaurus' global for <hr> */
  })}>{children}</div>
);

export default () => (
    <>
      <EuiCode>none</EuiCode>
      <Container>
        <EuiHorizontalRule margin="none" />
      </Container>
      <EuiCode>xs</EuiCode>
      <Container>
        <EuiHorizontalRule margin="xs" />
      </Container>
      <EuiCode>s</EuiCode>
      <Container>
        <EuiHorizontalRule margin="s" />
      </Container>
      <EuiCode>m</EuiCode>
      <Container>
        <EuiHorizontalRule margin="m" />
      </Container>
      <EuiCode>l (default)</EuiCode>
      <Container>
        <EuiHorizontalRule margin="l" />
      </Container>
      <EuiCode>xl</EuiCode>
      <Container>
        <EuiHorizontalRule margin="xl" />
      </Container>
      <EuiCode>xxl</EuiCode>
      <Container>
        <EuiHorizontalRule margin="xxl" />
      </Container>
    </>
);
```
