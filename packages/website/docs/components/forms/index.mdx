# Forms

## Form layouts
You can set the layout of your form controls and manage some of their behavior with some of these [form layout](./layouts/guidelines.mdx) components.

## Form controls
EUI provides a variety of data input components that enable a user to enter or select data. The components are divided into 5 different interaction types.
- [Text controls](./text/basic.mdx)
- [Numeric Controls](./numeric/basic.mdx)
- [Selection controls](./selection/overview.mdx)
- [Search and filter controls](./search-and-filter/search.mdx)
- [Date and time controls](./date-and-time/date-picker.mdx)
- [Other controls](./other/file-picker.mdx)
