# Typography

For most of your usages we recommend using [EuiText](../../../../components/display/text.mdx) or [EuiTitle](../../../../components/display/title.mdx) components directly **instead** of these theme tokens. Or head to the [text utilities](../../utilities/typography.mdx) page for helper classes and functions.

Hooks and tokens are provided for:

- [Font weight](./font_weight.mdx)
- [Font settings](./font_settings.mdx)
- [Font scale hook](./font_scale_hook.mdx)
