# Modal

**EuiModal** is the core overlay atom that provides focus management, keyboard navigation, and accessibility features for modal dialogs.

:::tip Looking for usage guidance?

For task-oriented recipes like confirmation dialogs, destructive actions, and decision flows, see the [Modal dialogs pattern](/docs/patterns/feedback-overlays/modal-dialogs).
:::

## Basic usage

Each **EuiModal** requires a specific set of nested child components. You can omit some if needed, but do not change their order.

:::warning Modals **can't** be dismissed by clicking outside

Unlike flyouts, modals can't be dismissed by clicking on the mask. This follows our modal usage guidelines, which require a primary action button, even if it just closes the modal.
:::

```tsx interactive
import React, { useState } from 'react';
import {
  EuiButton,
  EuiModal,
  EuiModalBody,
  EuiModalFooter,
  EuiModalHeader,
  EuiModalHeaderTitle,
  EuiText,
  useGeneratedHtmlId,
} from '@elastic/eui';

export default () => {
  const [isModalVisible, setIsModalVisible] = useState(false);

  const closeModal = () => setIsModalVisible(false);
  const showModal = () => setIsModalVisible(true);

  const modalTitleId = useGeneratedHtmlId();

  return (
    <>
      <EuiButton onClick={showModal}>Show modal</EuiButton>

      {isModalVisible && (
        <EuiModal aria-labelledby={modalTitleId} onClose={closeModal}>
          <EuiModalHeader>
            <EuiModalHeaderTitle id={modalTitleId}>
              Modal title
            </EuiModalHeaderTitle>
          </EuiModalHeader>

          <EuiModalBody>
            <EuiText>
              <p>Modal content goes here.</p>
            </EuiText>
          </EuiModalBody>

          <EuiModalFooter>
            <EuiButton onClick={closeModal} fill>
              Close
            </EuiButton>
          </EuiModalFooter>
        </EuiModal>
      )}
    </>
  );
};

```

## States

### Alert dialog

```tsx interactive
import React, { useState } from 'react';
import {
  EuiButton,
  EuiModal,
  EuiModalBody,
  EuiModalFooter,
  EuiModalHeader,
  EuiModalHeaderTitle,
  EuiText,
  useGeneratedHtmlId,
} from '@elastic/eui';

export default () => {
  const [isModalVisible, setIsModalVisible] = useState(false);

  const closeModal = () => setIsModalVisible(false);
  const showModal = () => setIsModalVisible(true);

  const modalTitleId = useGeneratedHtmlId();

  return (
    <>
      <EuiButton color="danger" onClick={showModal}>
        Show alert dialog
      </EuiButton>

      {isModalVisible && (
        <EuiModal 
          aria-labelledby={modalTitleId} 
          onClose={closeModal}
          role="alertdialog"
        >
          <EuiModalHeader>
            <EuiModalHeaderTitle id={modalTitleId}>
              Alert dialog
            </EuiModalHeaderTitle>
          </EuiModalHeader>

          <EuiModalBody>
            <EuiText>
              <p>This modal uses role="alertdialog" for critical actions.</p>
            </EuiText>
          </EuiModalBody>

          <EuiModalFooter>
            <EuiButton onClick={closeModal} fill>
              Close
            </EuiButton>
          </EuiModalFooter>
        </EuiModal>
      )}
    </>
  );
};
```

## Accessibility

- **Focus management:** Automatically traps focus within the modal
- **Keyboard navigation:** ESC key closes the modal (except alert dialogs)
- **Screen readers:** Announces modal title and role
- **ARIA attributes:** Supports `role="dialog"` and `role="alertdialog"`

## Design tokens

- **Background:** `euiTheme.colors.emptyShade`
- **Border:** `euiTheme.colors.borderBaseFloating`
- **Shadow:** `euiShadowXLarge`
- **Border radius:** `euiTheme.border.radius.medium`
