# Borders

```mdx-code-block
import { Example } from '@site/src/components';
```

The border tokens contain both individual border property values and full shorthand border properties.

## Types

```mdx-code-block
import { BorderPreview } from './borders/border_preview';
import { BordersTable } from './borders/borders_table';
```

<Example>
  <Example.Description>
    ### `euiTheme.border[type]` <Badge color="hollow">token</Badge>

    These common border types string together the base properties of color, width and style to form common full `border` properties.
  </Example.Description>
  <Example.Preview>
    <BorderPreview />
  </Example.Preview>
  <Example.Snippet>
    ```tsx
    css`
      border: ${euiTheme.border.thin};
    `
    ```
  </Example.Snippet>

## Color

EUI only has one base color it uses for all borders (or calculated borders).

```mdx-code-block
import { ColorPreview } from './borders/color_preview';
```

<Example>
  <Example.Description>
    ### `euiTheme.border.color` <Badge color="hollow">token</Badge>

    Color for all borders; Default is `colors.lightShade`

    You can apply this color directly to any `border-color` property.
  </Example.Description>
  <Example.Preview>
    <ColorPreview />
  </Example.Preview>
  <Example.Snippet>
    ```tsx
    css`
      border-color: ${euiTheme.border.color};
    `
    ```
  </Example.Snippet>
</Example>

<Example>
  <Example.Description>

  </Example.Description>
  <Example.Preview>
    <BorderWidthPreview />
  </Example.Preview>

