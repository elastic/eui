# Bottom bar

:::tip Looking for usage guidance?

For page-level action bars, positioning, displacement, and best practices, see the [Bottom bars pattern](../patterns/layouts/bottom-bars.mdx).

:::

::::tip
**EuiPageTemplate** offers a quick way to [apply a bottom bar to your page layouts](./page-template.mdx#with-bottom-bar).
::::

**EuiBottomBar** affixes a bar to the bottom of the viewport or a scroll container. This page focuses on API and minimal, isolated examples.

## Component

Like many of our other wrapper components, **EuiBottomBar** accepts a `paddingSize` prop, which can be set to `s | m (default) | l | none`.

```tsx interactive
import React, { useState } from 'react';
import { EuiBottomBar, EuiFlexGroup, EuiFlexItem, EuiButton, EuiButtonEmpty } from '@elastic/eui';

export default () => {
  const [showBar, setShowBar] = useState(false);

  return (
    <div>
      <EuiButton color="primary" onClick={() => setShowBar((show) => !show)}>
        Toggle bottom bar
      </EuiButton>

      {showBar && (
        <EuiBottomBar>
          <EuiFlexGroup justifyContent="flexEnd" gutterSize="s">
            <EuiFlexItem grow={false}>
              <EuiButtonEmpty color="text" size="s" onClick={() => setShowBar(false)}>
                Cancel
              </EuiButtonEmpty>
            </EuiFlexItem>
            <EuiFlexItem grow={false}>
              <EuiButton color="primary" fill size="s">Save</EuiButton>
            </EuiFlexItem>
          </EuiFlexGroup>
        </EuiBottomBar>
      )}
    </div>
  );
};
```

## Examples

### Positions

Use `position="fixed"` (default) for viewport-level bars, or `position="sticky"` to stick within a scroll container. You can also apply offsets with the `top | right | bottom | left` props.

```tsx interactive
import React from 'react';
import { EuiBottomBar, EuiText } from '@elastic/eui';

export default () => {
  return (
    <div css={{ overflow: 'auto', blockSize: 200 }}>
      <EuiText>
        <p>
          When scrolling within this example, the <strong>EuiBottomBar</strong>{' '}
          will stick to the bottom of scrollable container (with a 10px offset),
          but will not scroll with the page itself.
        </p>
      </EuiText>
      <div css={{ blockSize: 400 }} />
      <EuiBottomBar position="sticky" bottom={10}>
        <EuiText textAlign="center">
          <p>Scroll to see!</p>
        </EuiText>
      </EuiBottomBar>
    </div>
  );
};
```

### Displacement

`affordForDisplacement` (default `true`) reserves space to avoid overlapping page content.

```tsx interactive
import React, { useState } from 'react';
import {
  EuiBottomBar,
  EuiButtonGroup,
  EuiButton,
  EuiFlexGroup,
  EuiFlexItem,
} from '@elastic/eui';

export default () => {
  const [toggleIdSelected, setToggleIdSelected] = useState(null);

  const toggleButtons = [
    {
      id: 'bottomBarStandard',
      label: 'Show bottom bar',
    },
    {
      id: 'bottomBarWithoutAffordForDisplacement',
      label: 'Show bottom bar (without affordForDisplacement behavior)',
    },
  ];

  const onChange = (optionId) => {
    setToggleIdSelected(optionId);
  };

  return (
    <div>
      <EuiButtonGroup
        legend="Bottom Bar demo toggle buttons group"
        type="single"
        buttonSize="m"
        color="primary"
        options={toggleButtons}
        idSelected={toggleIdSelected}
        onChange={(id) => onChange(id)}
      />

      {toggleIdSelected && (
        <EuiBottomBar
          affordForDisplacement={toggleIdSelected === 'bottomBarStandard'}
        >
          <EuiFlexGroup justifyContent="flexEnd">
            <EuiFlexItem grow={false}>
              <EuiButton
                onClick={() => setToggleIdSelected(null)}
                color="text"
                size="s"
                iconType="cross"
              >
                close
              </EuiButton>
            </EuiFlexItem>
          </EuiFlexGroup>
        </EuiBottomBar>
      )}
    </div>
  );
};
```
