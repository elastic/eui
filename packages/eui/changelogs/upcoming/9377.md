- Added `container` prop to `EuiFlyout` for positioning flyouts relative to a container element instead of the viewport, enabling app-scoped flyouts that use `position: absolute` and container queries instead of `position: fixed` and viewport media queries ([#9377](https://github.com/elastic/eui/pull/9377))
- Added `container` prop to `EuiPortal` for mounting portal content into a specified container element instead of `document.body` ([#9377](https://github.com/elastic/eui/pull/9377))
- Added `container` to `EuiFlyout` component defaults configuration in `EuiProvider` ([#9377](https://github.com/elastic/eui/pull/9377))

- Fixed resizable flyout clamping to account for sibling flyout widths in side-by-side layout mode ([#9377](https://github.com/elastic/eui/pull/9377))
